# The Social Hive

Community events platform for Fullerton Cove residents.

> Movie nights, community events, bookings — all in one place.

## Stack

- **Frontend**: Next.js on Vercel
- **Database & Auth**: Supabase (PostgreSQL)
- **Deployment**: Auto-deploy from `main` branch via Vercel

## Status

🚧 In development — migrating from Element Movies (elementmovies.com.au)

## Project structure

```
/app          → Next.js app router pages
/components   → Shared UI components
/lib          → Supabase client, utilities
/public       → Static assets
```
